# Singleton Pattern Repository

Ce repository contient **différentes manières d’implémenter le Singleton Pattern** en Java, avec leurs avantages, inconvénients et considérations pour le **multithreading**.

---

## Qu’est-ce que le Singleton Pattern ?

Le **Singleton** est un **design pattern de création** qui limite l’instanciation d’une classe à **une seule instance**.  
Il est utilisé lorsque l’on souhaite un **objet global unique**, par exemple pour :  
- Gestion de configuration  
- Logger global  
- Connexion à une base de données  
- Gestion de cache
